# AI B2B Lead Finder + AI SDR

An AI-powered lead generation application that helps startups and sales teams discover potential B2B companies and generate personalized outreach messages automatically.

## Features
- B2B Lead Generation
- AI Cold Email Generator
- LinkedIn Outreach Message Generator
- Company Discovery by Industry and Location
- CRM-style Lead Management

## Inputs
- Industry
- Target Location
- Target Role
- Company Size

## Output
- Company Name
- Contact Email Format
- AI Cold Email
- LinkedIn Message

## Tech Stack
- Mattr AI
- Supabase
- AI Prompt Engineering

## Problem Solved
Manual lead generation is time-consuming for startups.  
This tool automates lead discovery and generates personalized outreach messages using AI.
